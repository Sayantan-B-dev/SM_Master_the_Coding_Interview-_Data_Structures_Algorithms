## Free Portfolio Templates – Resource Documentation

Based on the course materials, this document summarizes key recommendations for building a professional portfolio website using free templates.

---

### Why You Need a Portfolio

- **Must be live online** and shared with every job application.
- **1–2 BIG “wow” projects** are more important than the design itself.
- Every Zero to Mastery course includes at least one major project you can add to your portfolio.

> *“It’s more important what’s inside your portfolio than your actual portfolio.”*

---

### Recommended Free Template Sources

| Source | Description |
|--------|-------------|
| **[HTML5 UP](https://html5up.net/)** (favorite) | Responsive, modern HTML5 + CSS templates. Fully customizable, free for personal/commercial use. |
| **[Free CSS Templates](https://www.free-csstemplates.org/)** | Large collection of free website templates. |
| **[Cruip](https://cruip.com/)** | Free landing page templates built with Tailwind CSS and React. |
| **[Bootstrap Templates](https://getbootstrap.com/docs/5.3/examples/)** | Official Bootstrap examples (albums, dashboards, sign‑in pages, etc.) – great starting points. |

### Quick Build Options

- **AI tools** – Use an AI website builder (as shown in previous lessons) for fast, custom results.
- **Free templates** – Download, edit, and deploy (e.g., to GitHub Pages or Netlify).

### Key Reminders

1. **Host it online** – GitHub Pages, Netlify, Vercel, or your own domain.
2. **Showcase 1–2 big projects** – Quality over quantity.
3. **Link your portfolio** on LinkedIn, resume, and job applications.
4. **Keep it updated** as you complete more ZTM projects.

### Next Steps

- Choose a template from the list above.
- Replace placeholder content with your own projects, bio, and contact info.
